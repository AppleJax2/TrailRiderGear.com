# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-29

### Added
- Initial release of Astro SEO Blog Template
- Complete blog system with MDX support
- SEO optimization (sitemap, meta tags, Open Graph, schema.org)
- Dark mode with system preference detection
- Full-text search functionality
- Multi-language support (i18n ready)
- Dynamic categories and tags system
- Author profiles and multi-author support
- RSS feed generation
- Admin panel for content management
- Image optimization
- Responsive design with Tailwind CSS
- TypeScript support throughout
- Comprehensive documentation (README, FEATURES, CONTRIBUTING)
- GitHub issue and PR templates
- MIT License
- Example blog post

### Features
- Static site generation with Astro 5
- File-based content management
- Automatic sitemap generation
- Reading time calculation
- Related posts suggestions
- Social sharing buttons
- Table of contents generation
- Syntax highlighting for code blocks
- Comments system (optional)
- Analytics integration (Google Analytics)
- Deployment ready for Vercel, Netlify, Railway, etc.

### Developer Experience
- Hot module replacement
- Pre-commit hooks
- Automated build scripts
- Clear project structure
- Well-documented codebase
- Easy customization

## How to Update

To keep your fork up to date with the latest changes:

```bash
git remote add upstream https://github.com/Apatero-Org/astro-seo-blog-template.git
git fetch upstream
git merge upstream/main
```
